# PassGen

PassGen is a simple password generator made in Python. Because of its simplicity, it can run on pretty much any computer with Python installed. It uses the built-in string library, meaning you don't need a single extra library with Python installed.

# Requirements*
- Windows 7 x64 / Mac OS 10.9 Mavericks
- Python 3

*these are only if you are using the compiled binary
